# Rsbuild Project

## Setup

Install the dependencies:

```bash
pnpm install
```

## Get Started

Start the dev server:

```bash
pnpm dev
```

Build the app for production:

```bash
pnpm build
```

Preview the production build locally:

```bash
pnpm preview
```


### For development

- web: `npm run dev:web`
- electron: `npm run dev:electron`

### For production

- web: `npm run build:web`
- electron: `npm run build:electron`
